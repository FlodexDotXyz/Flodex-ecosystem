# Flodex

> Own a diversified portfolio of bonded Pump.fun coins in one transaction.

Flodex is a basket launchpad built on Solana that enables users to gain exposure to multiple bonded meme coins through a single transaction. Every basket is fully backed 1:1, transparent on-chain, and designed for liquidity and accessibility. The project centers on portfolio diversification rather than single-token speculation.

---

## 🚀 Overview

The meme coin ecosystem moves fast, but managing dozens of individual positions can be inefficient.

Flodex simplifies exposure by allowing users to:

* Buy multiple bonded assets in one transaction
* Access diversified meme coin baskets
* Verify backing on-chain
* Maintain liquidity across basket assets
* Reduce fragmentation across positions

Built on Solana, Flodex leverages high-speed, low-cost transactions to create a seamless basket investing experience.
